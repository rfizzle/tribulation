Tribulation

A comprehensive difficulty overhaul for Minecraft

Mobs grow stronger as you play longer, venture further, and descend deeper. Every hour survived raises the stakes with scaling stats, unlocked abilities, and brutal opt-in mechanics.

Download Latest Release

Core Systems

Four interlocking mechanics that transform Minecraft's difficulty curve from flat to exponential.

⚔

Mob Scaling

21 mob types scale across health, damage, speed, armor, toughness, and follow range. Per-mob tuning with independent rate and cap for each stat.

☠

Death Penalties

Lose 2 difficulty levels on death as relief. Shatter Shards drop rarely from scaled mobs and reduce your level by 5 when consumed.

❤

Hardcore Hearts

Opt-in permanent max health loss on death. Restore lost hearts with Heart Fragment items crafted from Shatter Shards.

👻

Soul Inventory

Opt-in inventory destruction on death. Only items enchanted with Soulbound survive. Forces deliberate gear choices.

Three Scaling Axes

Difficulty compounds from three independent sources. Each axis contributes a factor that multiplies mob attributes.

Time (Player Level)

Every hour of playtime advances your level (0–250). Higher level means mobs near you gain more stats. This is the primary axis.

Distance from Spawn

Beyond 1000 blocks from world spawn, mobs gain bonus health, damage, armor, and toughness. Caps at +150% of base stats. Overworld only by default.

Y-Level (Depth)

Below Y=62 or above it, mobs gain additional stats. Deeper caves and tall mountains both increase threat. Caps at +50%.

Quick Install

  1. 1 Install Fabric Loader 0.16.10+ for Minecraft 1.21.1
  2. 2 Install Fabric API (required dependency)
  3. 3 Drop tribulation-x.x.x.jar into your mods/ folder
  4. 4 Launch the game. Config generates at config/tribulation.json on first run
  5. 5 Adjust settings via the config file or /tribulation reload in-game

Requires Java 21. Remove any other mob scaling mods before installing.
